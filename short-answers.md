# Short Answers

## 1. Difference Between Google Knowledge Graph and Knowledge Panel

The Google Knowledge Graph is a structured database of entities and their relationships. It is used internally by Google to understand real-world objects like people, organizations, and places.

The Knowledge Panel is the visible search result box that displays information from the Knowledge Graph.

In summary:
- Knowledge Graph = database
- Knowledge Panel = visual output

---

## 2. How Google Determines Entity Identity

Google determines entity identity using:

- Structured data (Schema.org)
- Consistent branding across platforms
- Backlinks from authoritative sources
- sameAs social profile links
- Google Business Profile data
- User search behavior signals
- Content consistency across web properties

---

## 3. When to Create Custom Post Types Instead of Pages

Custom Post Types should be used when content is:

- repetitive in structure
- scalable
- dynamically categorized

Examples:
- testimonials
- portfolio items
- blog posts with categories
- job listings
- products

Pages are used for static content like About or Contact pages.
